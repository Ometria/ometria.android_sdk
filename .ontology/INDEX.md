# ometria.android_sdk

**Language**: Kotlin (Android)
**Purpose**: Official Ometria Android SDK. Published to Maven Central. Enables mobile apps to track customer behaviour events and receive push notifications via Firebase.

## Key capabilities
- Customer behaviour event tracking (screen views, basket events, orders, custom events)
- Firebase push notification handling
- Simple initialization: `implementation 'com.ometria:android-sdk:1.11.2'`
- Gradle dependency

## Code structure
- `OmetriaSDK/` — SDK source code
- `app/` — sample application
- `scripts/` — build/release scripts

## Distribution
- Maven Central: `com.ometria:android-sdk`

## Related repos
- `ometria.android_sdk_internal` — internal/development version
- `ometria.ios_sdk` — iOS equivalent
- `ometria.react_native_sdk` — React Native wrapper
